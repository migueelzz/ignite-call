## 🔥 Usage

1. Clone project:
   ```
   git clone https://github.com/migueelzz/ignite-call.git

   cd ignite-call
   ```
2. Install dependencies:
   ```
   npm install
   ```
3. Start project:
   ```
   npm run dev
   ```
   App running in `http://localhost:3000`.
